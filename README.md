# Cookie Crush
A simple chrome extension that automates cookie popups on websites.
Created for the **vsHacks 2026 hackathon**.

----

#### Providers Supported
Each of the following providers have beeen tested and are supported by Cookie Crush.
If a provider is not listed here, it may still work, but it has not been tested.

- OneTrust (https://www.onetrust.com/)
- Cookiebot (https://www.cookiebot.com/)
- YouTube (https://www.youtube.com/)
- Google (https://www.google.com/)
- Didomi (https://www.didomi.io/)
- StackOverflow (https://stackoverflow.com/)
- Quantcast (https://www.quantcast.com/)
- Tealium (https://tealium.com/)
- Usercentrics (https://usercentrics.com/)
- TrustArc (https://trustarc.com/)
- Drata (https://try.drata.com/product/gdpr)
- UniConsent (https://www.uniconsent.com/)
- WordPress (https://wordpress.com/)
- Sharethis (https://sharethis.com/)
